个人简历网站
========

背景：
-----
    因为听老师说在简历上有一个博客链接，或者个人网站链接会提高自己的简历质量，所以就决定自己实现一个较为简单的个人网页及后台服务。

环境：
-----
    Linux(centos7),火狐浏览器（其他浏览器可能不兼容）。
功能：
-----
    *为多个用户提供基本的登录、注册及修改密码等功能。
    *为用户提供一个存储，上传和下载的平台，方便用户进行多版本的简历管理
    *为用户提供一个个人网址方便展示比简历更加丰富的内容。
